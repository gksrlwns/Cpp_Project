### 미로 그리기

```
 플레이어를 만들고,
 시작지점과 끝지점을 정해서
 0.1초마다 플레이어가 시작점부터 끝지점까지 미로를 탙출하는걸 그릴거에요.
```

```
 1. 보드의 출발지점과 끝지점을 만들어줍니다.
 2. 플레이어의 position을 보드의 출발지점으로 만들어줍니다.
 3. 0.1초마다 플레이어가 아무방향이나 움직일수있게끔 세팅해준다.
  - (눈으로 움직이는걸 확인하기 위해서)
  - 벽으로는 이동하지 못하도록 설정
 4. 길찾기 알고리즘으로 길이동하도록 세팅
  // DFS, BFS, RightHand
  // 만약에 여러분들이 동굴이 갖혔는데, 출구를 찾아야한다.
  //  - 내 손을 오른쪽으로 뻗습니다.
  //  - 오른쪽에 벽이 안짚이면 오른쪽으로 회전하여 움직입니다.
  //  - 오른쪽에 벽이 짚이면 직진합니다.
  //  - 직진하다가 앞이 막혀있다. + 오른쪽도 막혀있다
  //  - 왼쪽으로 갑니다.
  //  - 왼쪽도 막혀있다. 그럼 뒤로가요.
 5. player Update에서 _path따라서 이동하도록 변경
```
