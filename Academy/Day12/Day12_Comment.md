### Red Black Tree
- 균등하게 높이를 조절해주는 트리
- 중앙 노드를 바꾸어주며 높이를 조절

### Map
- Red Black Tree 기술을 활용
-  데이터를 데이터 그자체로 활용하지 않고 key,value로 나눠 사용

```
10, 12, 13 의 경우
바이너리 -> 10 중앙, 우 12, 우 13 -> 높이 3
레드블랙 -> 12를 중앙노드, 10, 13을  좌우 노드로 배치 -> 높이 2
```
